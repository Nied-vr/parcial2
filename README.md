##1. Desarrollar un programa que determine si en una lista no existen elementos repetidos (dos ejemplos)

lista = [1, 2, 3, 4, 4, 5]
for n in lista:
    m = lista.count(n)
    if m >= 2:
        print(f"{n} Esta repetido en esta lista")
    else:
        print("Este numero no esta repetido ")
